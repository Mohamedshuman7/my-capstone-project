## Step 1: Pick Project Ideas

### Idea 1: Soccer/Sports Stat Tracker  
Track individual player stats and team standings.  
**Data Sources:** Football-Data.co.uk, Kaggle soccer datasets, API-Football.

### Idea 2: TV Show Rating and Review App (Chosen Project)  
A platform where users can create lists, rate, and review their favorite TV shows with a ranking system.  
**Data Sources:** TMDB API (ratings, show metadata), IMDb datasets.

Inspired by Letterboxd, this app focuses strictly on movies, allowing users to track watched and unwatched movies, rate them, and more.  
**Data Sources:** TMDB API, OMDb API for additional metadata.

## Step 2: Project Proposal: TV Show Rating and Review App

### Tech Stack  
- **Frontend:** React for a responsive, interactive web app  
- **Backend:** Node.js with Express for API and server logic  
- **Database:** MongoDB for storing user data, reviews, ratings, and lists  
- **APIs:** TMDB API for TV show metadata and ratings; IMDb datasets as supplemental data  
- **Hosting:** Vercel or Netlify for frontend; Heroku for backend // Subject to change, I am open to different ones as I get deeper into the project.

### Project Focus  
Full-stack development with an even focus on frontend UI and backend functionality to create a seamless user experience.

### Project Type  
Web application accessible on desktop and mobile browsers.

### Goal  
To provide TV show fans with a platform to browse shows, create custom watchlists, rate, and review shows, and see community rankings.

### User Demographic  
TV show enthusiasts of all ages seeking to organize and share their opinions, discover new shows, and engage with a community.

### Data and API Usage  
- Dynamic show data fetched from TMDB API  
- User-generated ratings, reviews, and lists stored in MongoDB  
- Potential challenges include API rate limits and data syncing

### Core Functionality  
- Browse and search TV shows  
- User authentication (signup/login)  
- Create, edit, and manage personalized lists  
- Rate and review TV shows  
- View aggregated ratings and rankings

### Stretch Goals  
- Social features: follow users, comment on reviews  
- Personalized show recommendations  
- Mobile-responsive design
